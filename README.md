# 📊 CSV Data Analysis Dashboard

A modern web-based CSV Data Analysis Dashboard built using **Flask**, **Pandas**, and **Matplotlib**. The application allows users to upload CSV files, perform statistical analysis, preview datasets, and generate visualizations through a clean and interactive interface.

---

## 🚀 Features

- 📁 Upload any CSV dataset
- 👀 Preview the first 10 rows
- 📈 Display dataset statistics
  - Number of Rows
  - Number of Columns
  - Missing Values
- 📊 Generate statistical summary
  - Mean
  - Standard Deviation
  - Minimum
  - Maximum
  - Quartiles
- 📉 Automatically generate a histogram for numerical data
- ⚠️ Error handling for invalid CSV files
- 💻 Clean and responsive user interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/NoorulHamdh1/CSV-Data-Analysis-Dashboard.git
```

### Navigate to the project

```bash
cd CSV-Data-Analysis-Dashboard
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
